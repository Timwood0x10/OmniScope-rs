## 📦 安装

```bash
git clone https://github.com/Timwood0x10/OmniScope.git
cd OmniScope
zig build -Drelease-fast
```

## 🚀 快速开始

```bash
# Python C extension
clang -emit-llvm -c mymodule.c -o mymodule.bc
./zig-out/bin/OmniScope mymodule.bc --boundary-only

# Go cgo
llgo -o mypkg.bc mypkg.go
./zig-out/bin/OmniScope mypkg.bc --json

# C++ FFI
clang++ -emit-llvm -c wrapper.cpp -o wrapper.bc
./zig-out/bin/OmniScope wrapper.bc --sarif
```

## 🎓 经验总结

### 成功因素

1. **架构复用**: 100% 复用现有 MemoryGraph + SemanticTree
2. **薄适配层**: 每个语言适配器 <200 LOC
3. **渐进交付**: 5个阶段，每个阶段可独立验证
4. **测试驱动**: 单元测试 + 集成测试 + 回归测试
5. **社区反馈**: Alpha/Beta 阶段收集反馈

### 风险与缓解

| 风险 | 影响 | 缓解措施 |
|------|------|---------|
| 时间超期 | 高 | 分阶段交付，P1功能可延后 |
| 性能下降 | 中 | 持续性能测试，优化热点 |
| FP率未达标 | 高 | 保守估计，boundary-only 兜底 |
| 语言适配器复杂度 | 中 | 从简单语言开始 (Python, C++) |
| 测试覆盖不足 | 中 | 自动化测试 + CI/CD |
